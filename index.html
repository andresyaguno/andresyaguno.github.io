Autor: Andres Yaguno (Primer Semestre - Computer Science)

Proyecto de aplicación de Álgebra Lineal y Teoría de Grafos.

El Modelo del Grafo Dirigido

Imaginemos una micro-web con 3 nodos donde:

  • El Nodo A enlaza a B y C.
  • El Nodo B enlaza solo a C.
  • El Nodo C enlaza de vuelta a A.

Cada iteración calcula el nuevo PageRank usando la fórmula con un factor de amortiguación $d = 0.85$:

A
0.33
B
0.33
C
0.33

Iteración actual: 0

Análisis Técnico para la Cátedra

Al hacer clic en el botón, el script resuelve el sistema de ecuaciones lineales de forma iterativa (método de las potencias). Verás cómo los valores de PageRank convergen gradualmente hacia un estado estacionario, demostrando las propiedades de una cadena de Markov irreducible y aperiódica.
